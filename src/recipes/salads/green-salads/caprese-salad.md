=== "Ingredients"
    | Amount | Ingredient                  |
    | :----- | :-------------------------- |
    | 450 g  | Mozzarella, roughly chopped |
    | 450 g  | Roma Tomatoes, chopped      |
    | 400 g  | White Beans, rinsed         |
    | 60 ml  | Pesto                       |
    | 45 ml  | Red Wine Vinegar            |
    | <1 ml  | Kosher Salt                 |
    | <1 ml  | Black Pepper                |

=== "Directions"
    Combine ingredients in a large bowl and serve.


!!! tip "Serves nicely with toasted focaccia."
