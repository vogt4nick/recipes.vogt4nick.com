# Farro

Farro refers to three hulled wheats: einkorn, emmer, and spelt. As external
recipes are often ambiguous, this page directs you to the correct grain.

| Grain                 | English | Continental | Italian       |
| :-------------------- | :------ | :---------- | :------------ |
| [Emmer](emmer.md)     | Farro   |             | Farro medio   |
| [Spelt](spelt.md)     |         | Farro       | Farro grande  |
| [Einkorn](einkorn.md) |         |             | Farro piccolo |
