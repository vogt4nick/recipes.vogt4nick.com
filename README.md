# Nick's E-Cookbook

A curated collection of recipes in multiple languages, organized and searchable,
built with [Zensical](https://zensical.org/).

**Live site:** https://github.com/vogt4nick/recipes.vogt4nick.com

This repository contains a personal recipe collection with recipes in English
and Austrian German. Each recipe follows a strict, consistent format to ensure
clarity, precision, and usability while cooking.
